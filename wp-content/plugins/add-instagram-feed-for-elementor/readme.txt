== Description ==

Add Instagram Feed For Elementor is a free plugin to display your Instagram posts. This plugin works with Elementor page builder. Furthermore, you can display the feed from your Instagram account in different layouts that come with this plugin.

= Additional Information =

Please click links below to find more information about the plugin.

[View Demo](https://demo.wpcharms.com/plugins/add-instagram-feed-for-elementor/) 
[Plugin Info Page](https://wpcapsules.com/item/add-instagram-feed-for-elementor/) 


== Key Features ==

* Display posts from your Instagram account.
* Select number of columns as per your requirement.
* Select 3 different image sizes.
* Show/hide like and comment counts.
* Enable/disable image overlay.
* Enable/disable border and shadow of images
* Square, circle or curve layout of images

== Installation ==

1. Go to Plugins > Add New and search for Add Instagram Feed For Elementor, install and activate it.
2. Go to Instagram Feed For Elementor menu at wp-admin dashboard of your site and setup Access Token. 
3. Go to Elementor page customization panel.
4. Scroll down and find WPCaps Elements.
5. You will find Instagram Feed added under WPCaps Elements.
3. Drag and drop Instagram Feed on the section you want to dispaly feed.


== Frequently Asked Questions ==

= What does this plugin needs to work ? =

This is an addon of Elementor page builder plugin so this plugin works if Elementor is installed and activated.

= Do this plugin works with all themes ? =

Yes, this plugin works with all themes.

= Can I add multiple layouts using this plugin ? =

Yes, you can use options available to create different layouts as per your need.
